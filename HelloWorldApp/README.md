# Hello World iPhone App

SwiftUI iOS app that shows a colorful “Hello, iPhone!” screen.

## Run it
1. Open the package in Xcode 15+ (`File > Open` and pick `HelloWorldApp/Package.swift`).
2. Select an iOS 17+ simulator (or your device after setting your team ID in `Package.swift`).
3. Press Run ▶ to launch.

## Customization
- Change the bundle ID or `teamIdentifier` in `Package.swift` to match your account.
- Edit `Sources/App/HelloWorldApp.swift` to tweak the gradient or text.
